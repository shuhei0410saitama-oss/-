# 📄 コンテンツ図解ツール Chrome拡張機能

Webページの内容を選択して、要点や構造を分かりやすく図解するChrome拡張機能です。

## 🎯 概要

記事やコンテンツを選択するだけで、以下の情報を自動的に抽出し、視覚的に図解します。

### 図解される内容

1. **📑 見出し構造**
   - h1〜h6の階層を視覚化
   - インデント表示で構造が一目瞭然

2. **📝 主な段落**
   - 段落ごとに整理して表示
   - 読みやすくまとめて提示

3. **📋 リスト項目**
   - 箇条書きリスト
   - 番号付きリスト

4. **🔑 キーワード**
   - 太字・強調されたテキスト
   - リンクテキスト
   - 重要な語句を抽出

5. **🖼️ 画像**
   - 画像とalt属性
   - タイトル情報

6. **📊 統計情報**
   - 文字数
   - 単語数
   - 見出し数・段落数

## 🚀 使用方法

### 1. 拡張機能の読み込み

1. Chromeで `chrome://extensions/` を開く
2. 「デベロッパーモード」をON
3. 「パッケージ化されていない拡張機能を読み込む」をクリック
4. `chrome-spec-generator` フォルダを選択

### 2. コンテンツの図解

1. 調査したいWebページを開く（例：ニュース記事、ブログなど）
2. 拡張機能アイコンをクリック
3. 「🎯 要素を選択」ボタンをクリック
4. ページ上の記事部分にマウスオーバー（青くハイライト）
5. 図解したい部分をクリック
6. 新しいタブで図解が表示されます

### 3. 図解結果の活用

- **📑 見出し構造**: 記事の構成を把握
- **📝 主な段落**: 内容を順番に確認
- **📋 リスト項目**: 重要なポイントを一覧
- **🔑 キーワード**: 主要トピックを把握
- **📋 内容をコピー**: テキスト形式でコピー

## 💡 使用例

### Yahoo!ニュースの記事を図解
1. Yahoo!ニュースの記事ページを開く
2. 拡張機能で記事部分を選択
3. 見出し、段落、キーワードが整理されて表示される

### Wikipediaの記事を図解
1. Wikipediaの記事を開く
2. 記事本文を選択
3. 見出し階層と内容が図解される

### ブログ記事を図解
1. ブログ記事を開く
2. 記事部分を選択
3. 構造と要点が視覚化される

## ⚙️ 技術仕様

- **Manifest Version**: V3
- **言語**: JavaScript（フレームワーク不要）
- **対応ブラウザ**: Google Chrome、Chromium系
- **権限**:
  - `activeTab`: 現在のタブへのアクセス
  - `scripting`: content scriptの注入
  - `storage`: 要素情報の一時保存

## 📁 ファイル構成

```
chrome-spec-generator/
├── manifest.json      # 拡張機能設定（Manifest V3）
├── popup.html/js      # ポップアップUI
├── content.js         # コンテンツ抽出スクリプト
├── background.js      # バックグラウンド処理
├── result.html/js     # 図解表示ページ
├── icons/             # アイコン画像
└── README.md          # このファイル
```

## 🔧 各ファイルの役割

- **content.js**: 見出し、段落、リスト、キーワード、画像を抽出
- **result.js**: 抽出した情報を視覚的に図解表示
- **popup.js**: 要素選択モードの開始/終了を制御

## 🐛 トラブルシューティング

### クリックできない場合
1. `chrome://extensions/` で拡張機能を再読み込み
2. Webページを再読み込み（F5）
3. F12で開発者ツールを開き、コンソールでエラーを確認

### 情報が表示されない場合
- 選択した要素に見出しや段落が含まれているか確認
- 記事全体ではなく、より大きな要素（記事コンテナなど）を選択

### 動作しないページ
- `chrome://` や `chrome-extension://` では動作しません
- 一般的なWebサイト（ニュース、ブログ、Wikipediaなど）で試してください
